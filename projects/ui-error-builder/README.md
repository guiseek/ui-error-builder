# UiErrorBuilder

## Install

### Yarn
```js
yarn add @guiseek/ui-error-builder
```
### npm
```js
npm i @guiseek/ui-error-builder --save
```

## Publishing

After building your library with `ng build ui-error-builder`, go to the dist folder `cd dist/ui-error-builder` and run `npm publish`.
